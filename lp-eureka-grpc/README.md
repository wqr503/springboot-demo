# springboot grpc eureka demo实战项目

#### 介绍
grpc 是一个高性能、开源和通用的 RPC 框架，面向服务端和移动端，基于 HTTP/2 设计。

eureka 是Netflix开源的一个RESTful服务，主要用于服务的注册发现。
